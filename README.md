# Project X App (Twitter-clone) - Next JS 13 - App router

### Techonology

- Next JS, Tailwind CSS, Firebase

### Main Features

- Auth (Sign-in & Sign-up)
- Create, Delete, Update Post
- Create, Delete, Update Comment
- Follow & Unfollow User
- Update User
- Fully Responsive

### How to run ?

- npm install
- npm run dev
